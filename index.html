Welcome to Javascript Tutorial

Important Notes!

  • JavaScript and Java are two different programming languages both in concept and design.
  • JavaScript is primarily used for client-side scripting in web browsers.
  • JavaScript can also be used on the server-side with environments like Node.js.
  • JavaScript was invented by Brendan Eich in 1995, and became an ECMA standard in 1997.
  • ECMA-262 is the official name of the standard. ECMAScript is the official name of the language.
